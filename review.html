This review MyFoxBadgeReader talks about the MyFox badge reader. I will tell you a bit more about the badge reader like the pros and the contras. Finally I will give you a conclusion what I find about the MyFox Reader.

Let us start with that I find about this badge reader. The first thing that is important for me is the User Interface, how it looks and how easily you can work with the device. There are a lot of wizards installed that make it easier for the user. It is a wireless device so you do not have to worry about some cables. It is a standard model but you recognize the symbols on the device like lock, unlock, home and settings. People who have a device like an Xperia z2/z3 can connect to this device. Of course you as the owner of the device can choose if you want it to connect or not.

Then some things I definitely do not like about the device. The options menu for the users is still in French. I hope it will get an update as soon as possible so it has the main language in the options menu. A little bit of a downside it is a fairly standard model it maybe looks kind of boring.
