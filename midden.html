FAQ

Comment est-ce que je peux changer la langue du correcteur orthographique sur Google Docs?
Quand vous êtes dans Google Docs, vous voyez 'fichier' dans la barre de menu. Sous 'fichier' vous cliquez sur 'langue' et vous choisissez votre langue préférée.

J'ai supprimé un document, comment est-ce que je peux remettre la document?
D'abord vous allez aux https://drive.google.com/drive/trash. Puis vous faites un clique droit sur le document et vous pouvez le remettre. Finalement vous cliquez sur 'récupérer'.

Comment peut-on stocker un document sur Google?
Google utilise un système dans lequel le document est stocké automatiquement. Donc, vous fermez le document. Google stocke le document pour vous.

Comment est-ce que j' ajoute un autre compte sur Google?
Quand vous vous déconnectez sur Google, vous pouvez cliquez sur 'ajouter personne'. Apres vous completez vos cordonnées. Maintenant vous pouvez choisir entre plusieurs comptes.

Je peux me déconnecter où?
D'abord vous devriez cliquez sur votre nom à droite en haut. Après vous pressez sur 'échange de personne'. Ensuite vous cliquez sur 'options' et vous sélectonnez 'retirer de personne'.
