GNS3

Zo kan je visueel je plan, test en troubleshoot op elk platform testen zonder de nodige hardware. Je kan alle virtuele interfaces testen om een reële representatie te showcasen. Dit wordt ondersteund door elk platform, de voornaamste platforms zijn Windows, Linux en Mac. GNS download -> GNS3 Download . Ik heb natuurlijk niet alleen gewerkt aan dit project. Dit project heb ik verwezenlijkt met mijn 2 collegas, Kenzo (Kenzo zijn website) en Kathleen (Kahtleen haar website)

bblablalba

Eerst moet je een image downloaden dan kan je deze installeren. Dit kan je doen in GNS3 Edit -> Preferences -> IOS-routers -> New. Dan selecteer je de image en wordt die geïmporteerd
Nu kan je allerlei dingen proberen in het kleine netwerk dat je gemaakt hebt, pingen, console openen,…
Hier kan je zien dat de test die wij probeerden niet werkte.
